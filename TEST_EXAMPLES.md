# 🧪 테스트 문구 모음

업데이트된 AI 에이전트를 테스트하기 위한 예제 문구들입니다.

## 1️⃣ 번역 기능 테스트

번역 모드로 자동 전환되는 문구들:

```
이 문서 번역해줘
번역해줘
번역 부탁
이 파일 번역
translate this
```

**파일과 함께 사용:**
```
이 파일 번역해줘
(파일 업로드 후)
```

## 7️⃣ 통합 테스트 시나리오

### 시나리오 1: 번역 → 일반 질문
```
1. "이 파일 번역해줘" (파일 업로드)
2. "번역된 내용을 요약해줘"
```

### 시나리오 2: 연구 → 상세 질문
```
1. "LangGraph 연구해줘"
2. "LangGraph와 LangChain의 차이점은?"
```

### 시나리오 3: 파일 분석 → 추가 질문
```
1. "이 파일 분석해줘" (파일 업로드)
2. "이 문서의 핵심 키워드는?"
```

## 🚀 빠른 테스트 명령어

### Backend 실행 확인
```bash
# 가상환경 활성화 후
cd backend
uvicorn app.main:app --reload
```

### API 테스트 (curl)
```bash
# 일반 질문
curl -X POST "http://localhost:8000/agent" \
  -H "Content-Type: application/json" \
  -d '{"question": "Python이란?"}'

# 연구 모드
curl -X POST "http://localhost:8000/agent" \
  -H "Content-Type: application/json" \
  -d '{"question": "AI 에이전트 프레임워크 연구해줘"}'

# Health Check
curl "http://localhost:8000/health"
```

### Streamlit 테스트
```bash
# 가상환경에서
streamlit run streamlit_app.py
```
